# HypeToken
HypeToken es una plataforma web de fidelización gamificada para negocios: reemplaza tarjetas físicas por tokens, retos y recompensas, permitiendo a los clientes consultar su estado sin instalar apps. ArenaCoins es el primer demo (Oasis Games) que muestra el flujo completo de consulta de puntos, niveles/clanes y beneficios.
